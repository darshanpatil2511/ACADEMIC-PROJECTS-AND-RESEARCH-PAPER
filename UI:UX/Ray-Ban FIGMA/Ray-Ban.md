Ray-Ban Mobile App – High-Fidelity UI/UX Prototype:- 

A premium, end-to-end mobile shopping experience designed for Ray-Ban, featuring dark-mode aesthetics, seamless navigation, powerful product discovery, and specialized eyewear features such as AI Try-On, Lens Power Selector, Store Locator, Repairs & Services, and full e-commerce flows.

This project was created as part of the UI/UX Design Final Project, built entirely in Figma with AI-assisted design workflows.


Overview:- 

This project is a high-fidelity mobile app prototype built for Ray-Ban, showcasing a complete e-commerce eyewear experience from onboarding to checkout.
The design emphasizes:

Premium dark-mode UI

Ray-Ban brand personality

60–30–10 color design system

Modern UX patterns

Micro-interactions and component variants

Clean information architecture

Realistic service flows (Repairs, Returns, AI Try-On)

Project Objectives:- 

Created an end-to-end mobile retail app for Ray-Ban with a polished high-fidelity UI.

Ensure complete user flows including authentication, browsing, filtering, cart, checkout, order tracking, returns, repairs, and service features.

Leverage Figma Components, Variants, Auto-Layout, Nested Frames, and Smart Animate for a scalable, reusable design system.

Integrate Figma AI to accelerate screen generation, copy systems, and consistency.

Demonstrate excellent UX thinking, onboarding strategy, accessibility, and visual design.

Need clarity on repairs and warranty

Prefer dark themes for premium brands
